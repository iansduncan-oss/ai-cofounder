       - `PUT /api/settings/budget` — accepts `{ dailyUsd: number, weeklyUsd: number }`, validates non-negative, upserts both keys
       - Register in server.ts alongside other route plugins

    2. Update `apps/agent-server/src/routes/usage.ts` `/budget` handler:
       - Read `daily_budget_usd` and `weekly_budget_usd` from `getAppSetting()` first

       - This ensures dashboard-saved thresholds take effect immediately

    3. Update `apps/agent-server/src/services/budget-alert.ts` `checkBudget()`:

       ```typescript
         weeklyUsd: number;
       }
       ```

       ```typescript
         return this.fetch<AppSettings>("/api/settings");

           method: "PUT",
           body: JSON.stringify(data),
         });
       }
       ```
