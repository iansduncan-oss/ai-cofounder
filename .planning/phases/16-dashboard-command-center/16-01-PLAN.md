       - `getAppSetting(db, key: string): Promise<string | null>` — single row lookup
       - `upsertAppSetting(db, key: string, value: string): Promise<void>` — insert or update on conflict
       - `getAllAppSettings(db): Promise<Record<string, string>>` — returns all settings as key-value map

    3. Export new functions from `packages/db/src/index.ts`

    4. Create migration `packages/db/drizzle/0022_add_app_settings.sql`:
       ```sql
       CREATE TABLE IF NOT EXISTS "app_settings" (
         "key" text PRIMARY KEY NOT NULL,
         "value" text NOT NULL,
         "updated_at" timestamp DEFAULT now()
       );
       ```

    5. Create `apps/agent-server/src/routes/settings-api.ts`:
       - `GET /api/settings` — returns `{ dailyBudgetUsd: number, weeklyBudgetUsd: number }` by reading `daily_budget_usd` and `weekly_budget_usd` keys from appSettings, falling back to optionalEnv values
       - `PUT /api/settings/budget` — accepts `{ dailyUsd: number, weeklyUsd: number }`, validates non-negative, upserts both keys
       - Register in server.ts alongside other route plugins

    6. Update `apps/agent-server/src/routes/usage.ts` `/budget` handler:
       - Read `daily_budget_usd` and `weekly_budget_usd` from `getAppSetting()` first
       - Fall back to `optionalEnv("DAILY_BUDGET_USD", "0")` and `optionalEnv("WEEKLY_BUDGET_USD", "0")` if DB returns null
       - This ensures dashboard-saved thresholds take effect immediately

    7. Update `apps/agent-server/src/services/budget-alert.ts` `checkBudget()`:
       - Same pattern: read from DB first, env fallback
       - Pass `db` instance (already available in the service constructor or via the worker context)

    8. Add to `packages/api-client/src/types.ts`:
       ```typescript
       export interface AppSettings {
         dailyBudgetUsd: number;
         weeklyBudgetUsd: number;
       }
       export interface UpdateBudgetInput {
         dailyUsd: number;
         weeklyUsd: number;
       }
       ```

    9. Add to `packages/api-client/src/client.ts`:
       ```typescript
       async getSettings(): Promise<AppSettings> {
         return this.fetch<AppSettings>("/api/settings");
       }
       async updateBudgetThresholds(data: UpdateBudgetInput): Promise<void> {
         await this.fetch<void>("/api/settings/budget", {
           method: "PUT",
           body: JSON.stringify(data),
         });
       }
       ```
