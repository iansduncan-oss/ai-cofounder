    **1. Create use-auth.ts hook (apps/dashboard/src/hooks/use-auth.ts):**
    New file with in-memory token management:
    ```typescript
    // Module-level token — survives React re-renders, cleared on page reload
    let _accessToken: string | null = null;

    export function getAccessToken(): string | null {
      return _accessToken;
    }

    export function setAccessToken(token: string | null): void {
      _accessToken = token;
    }
    ```

    Export `useAuth()` hook with:
    - `isAuthenticated`: derived from `!!_accessToken`
    - `login(email: string, password: string)`: POST to `/api/auth/login` with `credentials: 'include'`, parse `{ accessToken }`, call `setAccessToken(accessToken)`
    - `logout()`: call `setAccessToken(null)`, POST to `/api/auth/logout` with `credentials: 'include'`, redirect to `/dashboard/login`
    - `refresh()`: POST to `/api/auth/refresh` with `credentials: 'include'` (cookie sent automatically), parse `{ accessToken }`, call `setAccessToken(accessToken)`, return the token. On failure, call `setAccessToken(null)` and return null.

    Use `useState` or `useReducer` to trigger re-renders when token changes. The module-level `_accessToken` is the source of truth; the React state mirrors it for reactivity.

    The base URL for fetch calls should use `import.meta.env.VITE_API_URL || ''` (same as existing client.ts).

    **2. Update api/client.ts (apps/dashboard/src/api/client.ts):**
    Replace the current static token approach:
    ```typescript
    import { ApiClient } from '@ai-cofounder/api-client';
    import { getAccessToken, setAccessToken } from '@/hooks/use-auth';

    const baseUrl = import.meta.env.VITE_API_URL || '';

    export const apiClient = new ApiClient({
      baseUrl,
      getToken: getAccessToken,
      onUnauthorized: async () => {
        // Silent refresh using HttpOnly cookie
        const res = await fetch(`${baseUrl}/api/auth/refresh`, {
          method: 'POST',
          credentials: 'include',
        });
        if (!res.ok) {
          setAccessToken(null);
          window.location.href = '/dashboard/login';
          return null;
        }
        const { accessToken } = await res.json() as { accessToken: string };
        setAccessToken(accessToken);
        return accessToken;
      },
    });
    ```
