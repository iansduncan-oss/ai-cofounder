## Gaps Summary

One blocking gap prevents full TERM-02 achievement: the `workspaceService` parameter is not forwarded to `CoderAgent` inside `TaskDispatcher.constructor()`.

**The gap in full:**

`apps/agent-server/src/agents/dispatcher.ts` line 77 reads:
```typescript
["coder", new CoderAgent(registry, db, sandboxService)],
```

It should read:
```typescript
["coder", new CoderAgent(registry, db, sandboxService, workspaceService)],
```

The `workspaceService` is accepted as a constructor parameter (line 72) and correctly forwarded to `DocWriterAgent` (line 81). The omission for `CoderAgent` means that when the autonomous executor calls `dispatcher.runGoal()`, the CoderAgent never gets the `read_file`, `write_file`, or `list_directory` tools — it can only generate code as text, not write files to disk.

- ApiClient has 3 new methods and 2 new types
