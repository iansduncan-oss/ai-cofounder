      5. If decision found, generate embedding via `this.embeddingService.embed(decision)` (if available).
      6. Save via `saveMemory(db, { userId, category: "decisions", key: title, content: decision, importance: 8, metadata: { rationale, alternatives, conversationId, extractedAt: new Date().toISOString() }, embedding })`.
      7. Log at info level: "decision auto-extracted" with title and conversationId.
